# The `C` version of The P game

To try this at home:
1. Be sure u are using Linux system
2. Install `sdl2` via `sudo apt-get install libsdl2-dev`
3. Do `git clone https://github.com/vovakirdan/the-cp.git`
4. Do `cd the-cp`
5. Do `make`
6. Do `./the-cp`

As easy as can be!
